# 🎓 Udemy Clone (HTML & CSS)

A simple Udemy homepage clone built using HTML and CSS.  
This project is for learning frontend design and layout.

## 🚀 Features
- Responsive Navbar
- Course Cards UI
- Banner Section
- Categories Section
- Footer

## 🛠️ Tech Used
- HTML5
- CSS3 (Flexbox & Grid)

## 📂 Project Structure
udemy-clone/
│── index.html
│── style.css
│── images/

## ▶️ How to Run
1. Download or clone the repo
2. Open `index.html` in your browser

## 📸 Preview
Simple UI similar to Udemy homepage

## 📌 Note
This is just a UI clone, no backend functionality included.
